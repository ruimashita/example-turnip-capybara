# インストール

## install qt
ref: https://github.com/thoughtbot/capybara-webkit/wiki/Installing-Qt-and-compiling-capybara-webkit

```
brew install qt # osx

sudo apt-get install libqt4-dev # ubuntu
```


# bundle install
```
bundle install --path vendor/bundle
```

# テスト実行
```
bundle exec rspec 
```
